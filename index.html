HTML to remember

Used in Name tag usage notes attributes
Tables table <table> Outer tag of table
table head <thead> Head of the table. Should contain a <tr> with <th>'s inside
table body <tbody> Body of the table. Should contain <tr>s for the rest of the rows.
table footer <tfoot> Footer of the table. Should contain a <tr>s for some summary of the table.
table caption <caption> Caption for the table.
table row <tr> A row of the table. Should contain (a) <th> and some <td> for the rest of the columns.
table header <th> The start of a row or a column. Explains what is contained in the row/column. scope: is it the header of a "column" or "row"
colspan/rowspan: how many columns/rows does it span, default 1.
table data <td> The data contained in a cell. colspan/rowspan: how many columns/rows does it span, default 1.
